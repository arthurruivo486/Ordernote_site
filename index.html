Como instalar (Android)

1. Baixe o arquivo APK usando o botão acima.

2. Abra o arquivo no seu dispositivo Android.

3. Se necessário, ative "Fontes desconhecidas" em Configurações > Segurança.

4. Siga as instruções de instalação.

5. Abra o aplicativo e comece a usar!

Privacidade e Segurança

O Ordernote armazena todos os dados localmente no seu dispositivo. Não coletamos nenhuma informação pessoal.

Recomendamos fazer backup regular dos seus dados para evitar perdas.

Para atualizações, desinstale a versão anterior antes de instalar a nova.
